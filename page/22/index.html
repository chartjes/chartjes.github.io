# Learn to code like a Grumpy Programmer Welcome to my course where I wan to teach you how to write PHP code more like a Grumpy Programmer by teaching you what I think are the building block skills you will need going forward. In this course I am going to teach you: * testing techniques and patterns specific to PHP * using automated code formatters for consistency * how static code analysis helps prevent problems while refactoring * using mutation testing to verify how effective your tests are.
