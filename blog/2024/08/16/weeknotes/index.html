Weeknotes August 16, 2024
post

What I Did

  • played my IBL games via instructions (team managers create documents for how they want their team to be run)
  • finished writing all the tests for additional scenarios for the transaction processing job
  • no Magic: The Gathering this week as Commander night was my 26th wedding anniversary
  • stomach is feeling a little better -- I wonder if the ache I feel is because of damage all that stomach acid has done :(

Longer Threads

I came across a blog post from a developer about why they switched to Emacs and it make me wonder if I should try switching from Vim to Emacs. Again.

I tried it once before but couldn't duplicate my developer work flow. I feel like the combination of Emacs + xah-fly-keys is a great start since it would give me a similar modal editing experience that I expect in my editor.

Of course, the bigger issue is that I would have to set things up twice -- once on my work computer that ZiffDavis gave me and once on my own machine. I also pay for a license for PhpStorm and it is up for renewal soon.

Good to keep learning things. I've been using Vim so long that navigating via H-J-K-L is deeply embedded in muscle memory. I have noticed over the years that I don't really do much "web surfing" on my laptop -- I use my iPhone for that so having Vim bindings for my browser doesn't make mush sense.

Everything Else

  • if you are interested in ergonomic keyboards I would check out the Glove80 from moergo
