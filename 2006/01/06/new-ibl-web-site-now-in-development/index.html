New IBL web site now in development post

Yes, it's finally up and running. I settled on Drupal as the CMS of choice and spent some time hacking together a module to automatically display the latest standings. It took me about an hour from starting the tutorial to getting it to work. Right now it's done both as a "block" that's on the side of the page and as a standalone page with expanded information about the standings. Go take a look:

  • main page
  • expanded standings page

Obviously, there is more work to be done but we've started.
