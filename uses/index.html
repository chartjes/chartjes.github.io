What Chris Uses

(Last updated December 10, 2023)

A common question a lot of developers get about almost anything they are doing is "what are you using?". In that spirit, here is a list of things I am currently using.

Hardware

  • 2021 Apple MacBookPro with M1 Max chip. 64 GB RAM, 1 TB Drive
  • Apple Studio Display monitor
  • Kinesis Advantage2 LF ergonomic keyboard with an upgraded open-source controller
  • Ploopy Classic Trackball
  • Apple iPhone 13

I have been an Apple user for a very long time. Whenever it is time for me to get a new computer I inevitably ask myself if I should switch (usually to one running Linux). Then I remember what a huge pain in the ass it would be to go back to Linux, even though it's suport for Docker is so great.

So I stay in the Apple ecosystem. I still think the hardware is very nice. My Studio Display has been the best monitor I have ever owned. Great picture quality, really good sound from it and the built-in web cam is perfectly suited to the regular video meetings I have to do. For the most part, I stick to stable versions of everything I use and it all literally just works. Sorry h8rs, I like Apple's stuff.

The keyboard and trackball are all part of my plan to stay ahead of any RSI problems. So far, so good, although it has wrecked my ability to get any real work done on the laptop when I travel.

Software

This is for my day-to-day programming work

  • PhpStorm as my day-to-day IDE for PHP
  • NeoVim for everything else that is not PHP
  • Homebrew to manage OS-level dependencies
  • Warp as my terminal
  • Alfred as my launcher/helper
  • Safari Technology Preview as my browser
  • Fish as my shell
  • I have been using GitKraken as a Git client
  • Joplin to take notes for my Zettelkasten system
  • This blog is generated using Sculpin

Evaluating

Of course, I am always trying stuff out.

Atuin

I spend a lot of time in shells, so I am giving Atuin a trial to see how helpful it is for locating older shell commands.

Home Row Mods

Now that I have a keyboard controller that uses an open source firmware I can experiment with some keyboard layouts in a way that wasn't possible with Kinesis' proprietary controller for my keyboard. I have updated my keyboard layout to make use of home row mods to make my typing a little more efficient.

The biggest change has been trying to re-map muscle memory so that I use the home row mods for shift and control. It will take some time but I will get there.

Amaethyst

I was finding my desktop was getting cluttered while working so to reduce distractions I decided to try out a tiling window manager. The first one I evaluated was Yabai but after a while I was having issues getting the window layouts just the way I want. I have now switched to Amaethyst.
