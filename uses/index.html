What Chris Uses

(Last updated December 27, 2024)

A common question a lot of developers get about almost anything they are doing is "what are you using?". In that spirit, here is a list of things I am currently using.

Hardware

  • 2021 Apple MacBookPro with M1 Max chip. 64 GB RAM, 1 TB Drive
  • Apple Studio Display monitor
  • Dygma Defy split ergonomic keyboard
  • Ploopy Classic Trackball
  • Apple iPhone 15 Plus
  • over-the-top Ubiquiti home network setup that I have been running since 2017

Only change is I switched from my QMK-modified Kinesis Advantage 2 LF keyboard. Still staying in the Apple ecosystem.

Software

This is for my day-to-day programming work

  • Sequoia 15.1.1
  • PhpStorm as my day-to-day IDE for PHP via paid subscription
  • PyCharm for some Python coding at work
  • NeoVim for everything else (like this document)
  • Homebrew to manage OS-level dependencies
  • Ghostty as my terminal emulator
  • Alfred as my launcher/helper
  • Safari as my browser (I use Chrome at work because they only support it)
  • Atuin for shell history and searching
  • Docker for containerization via Podman
  • This blog is generated using Sculpin

Since the last time I updated, here are the changes:

  • back to stock Safari
  • went from Warp to Kitty to Ghostty for terminal emulator
  • stopped using home row mods
  • Docker Desktop to Podman

Evaluating

Of course, I am always trying stuff out.

Tiling window manager

I went with Yabai and then Aerospace and then found out that I could do 99% of what I want with changes in MacOS Sequoia.

Sketchybar

The idea of replacing the usual MacOS status bar at the top with a custom one is something I find interesting and that is where Sketchybar comes in
